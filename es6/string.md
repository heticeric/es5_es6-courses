# Ton template en `String`

```js
const name = "Eric";
const welcome = `Salut ${name}, Good Morniiiiiiiiiiiing!`;

console.log(welcome);

// Works well with multi-line strings
const message = `
  # Title

  This is a multi line string as markdown.
  It's pretty nice.
`;
console.log(message);
```
